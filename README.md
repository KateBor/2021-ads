Задачи с informatics-msk: https://informatics.msk.ru/mod/statements/view.php?id=64802#1

В задачах на реализацию структур данных и выполнение команд на ней подразумевается, что вы не будете использовать Stack/Queue из Java. В "олимпиадных" задачах можно использовать стандартные реализации java.util.Queue какие найдете, java.util.Stack использовать нельзя.

За каждое полностью рабочее решение дается 2 балла (да, даже за первую задачу).
